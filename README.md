# Estancia de Investigación - Programa Delfín

Este repositorio está destinado a compilar, organizar y respaldar todos los proyectos, códigos, análisis de datos y trabajos académicos desarrollados durante el **Verano de la Investigación Científica y Tecnológica del Pacífico (Programa Delfín)**.

El objetivo principal de este espacio es mantener un registro claro, limpio y estructurado de los avances, protocolos de software, procesamiento de datos y documentación técnica generados a lo largo del periodo de la estancia.

---

## 📂 Estructura del Repositorio

Para mantener el proyecto organizado de manera académica y profesional, el código y los entregables se estructuran de la siguiente forma:

* `src/` — Código fuente principal, scripts de desarrollo, módulos y algoritmos de software. Mas como todo aquello que no sea un .ipynb 
* `data/` — Protocolos de procesamiento, limpieza y análisis de datos (scripts automatizados).
* `docs/` — Documentación técnica, reportes de avance, artículos de referencia y ensayos.
* `notebooks/` — Cuadernos interactivos para análisis exploratorio de datos, visualizaciones y pruebas iniciales. aquí podemos poner los .ipynb
